## 一个简单的pdf压缩脚本

## Required
Python3

## Mac上使用：

1. 安装相关
```
brew install ghostscript
```

2. 使用脚本
```
# 压缩单个PDF
python/python3 pdfc.py -f {file_name}
# 批量压缩PDF
python/python3 pdfc.py -r {folder_path}
```

## Windows使用

1. 安装相关
[Ghostscript](https://www.ghostscript.com/download/gsdnld.html)


2. 使用脚本
```
# 压缩单个PDF
python/python3 pdfc.py -f {file_name}
# 批量压缩PDF
python/python3 pdfc.py -r {folder_path}
```
